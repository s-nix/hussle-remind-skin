# Hussle Skin Reminder Webapp

A lightweight webapp that reminds Hussle to rotate his Doctor Strange skin in Marvel Rivals.

## Run

1. Open this folder in VS Code.
2. Start a local static server from this folder, for example:

   ```powershell
   python -m http.server 8080
   ```

3. Open http://localhost:8080 in your browser.

You can also open `index.html` directly, but browser notifications are more reliable when served from localhost.

## Features

- Adjustable reminder interval from 5 to 120 minutes.
- Manual Remind Now action.
- Start and Stop reminder scheduling.
- Optional browser notifications.
- Live countdown to next reminder.
- Reminder activity log with clear action.
